Let’s Enjoy Scala

Scala 2.10.1-RC3がリリースされました

Created by moors on 2013-03-06. Updated: 2013-03-07, 00:28 (翻訳掲載 2013-03-09 mhanada)

Scala 2.10.1のリリース候補第三弾を発表できてうれしく思います。

Scala開発チームと貢献者により2.10.0から189の不具合が修正されました。全部で249のプルリクエスト(RC3では+4)がGitHubで公開され、そのうち225がテストとレビューを経てマージされました(RC3では+4)。

2.10.1-RC3を試していただくようお願いします。これは2.10.0を単純に置き換えられるように設計されています。ただし、（まれな）ケースですが大きな変更をするよりバグを修正しない方がよいと判断した場合は例外です（今のところこれに該当するのは1件だけです）。2.10.0にはなかった深刻な問題についての報告はぜひお寄せください。正式リリースまでに修正するよう努めます。

このリリース候補はリリースから1週間以内に重大な問題がなければ正式版となる予定です。

既知の問題

バグを報告する前に既知の問題についてご確認いただくようお願いします。

  • Eclipse 3.7(Indigo)用
  • Eclipse 3.8/4.2(Juno)用（このバージョンのサポートは試験的なものです）

Scala 2.9.3 がリリースされました

Created by moors on 2013-02-28. Updated: 2013-02-28, 09:02 (翻訳掲載 2013-03-08 mhanada)

Scala2.9.xのメンテナンスシリーズ 2.9.3の正式版を発表できることをうれしく思います。

このリリースには以下の改善が含まれています。

  • SIP-14の実装の2.9へのバックポート(訳注:2.10で導入された新しいFutureとPromise)
  • 安定性と応答性を改善するためにScala IDEから受け継いだ数多くの修正
  • インクリメンタルコンパイルをさらに早くするためのコンパイラの修正

Scala IDE for Eclipse

Scala 2.9.3用のScala IDE 3.0-RC1を以下のアップデートサイトからインストールできます。

  • Eclipse 3.7(Indigo)用
  • Eclipse 3.8/4.2(Juno)用

Created by heathermiller on 2013-02-27. Updated: 2013-02-27, 03:03 (翻訳掲載 2013-03-08 mhanada)

Scala WorkshopはScalaやScalaに関連する研究、開発に関する重要なフォーラムです。一日通して行われる研究発表やツールのデモにぜひご参加ください。Scalaを使っている、あるいはScalaに興味を持っている研究者や開発者、学生たちと交流しましょう。Scala Workshopは産業と学問の間にある溝を埋めるためのプログラミング言語やソフトウェア工学における最先端の成果を知る場です。

Functional Programming Principles in Scala – Scalaで学ぶ関数型プログラミングの原則: 感想と統計

Created by heathermiller on 2012-12-20. Updated: 2012-12-21, 23:42 （翻訳掲載 2013-01-20 mhanada）

この記事では、人気のあるMOOC（公開オンライン講座）「Functional Programming Principles in Scala」の公開を通じて得た知識について述べるとともに、この講座にどのような人が参加し、どのように受講し、またどのような感想を持ったかについて考察します。これらの多くの統計を数々のインタラクティブなチャートで可視化していますが、さらにこれらのデータととともにプロットを生成するコードを公開していくことにしました。ScalaベースのプロジェクトでScalaによる関数プログラミングを使って楽しく統計を扱うことができます。HTML/JavaScriptを生成するので、簡単に可視化と共有ができます。あなたの発見をぜひ私たちと共有してください。今後の記事でみなさんのプロットをたくさん共有していきます。こちらの記事をご覧ください!(英語)

(訳注:Functional Programming Principles in ScalaはCourseraというオンライン教育サービスで公開されている講座です。Martin Odersky教授がビデオで講義してくれます。もちろん英語ですが、ビデオ内で使用されているスライドのPDFや字幕ファイルなども提供されていますのでヒアリングの練習にもなりそうです。ありがたいことに無料で受講できますのでチャレンジしてみてはいかがでしょうか。ちなみに先の記事によると日本でも45名の方が受講されたようです。）
